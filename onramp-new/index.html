Gemini API Key (Optional)

ai.google.dev/gemini-api/docs/api-key

Get an API Key

  1. Click "Get an API key" in Google AI Studio.
  2. Select your Google Cloud project.
  3. Click "Create API key in existing project".
  4. Copy the key (usually starts with AIza...).

Set Env Variable

# Set your key
export GEMINI_API_KEY="AIzaSyA..."

# Verify
gemini
